### Заметки:
- Для КП выбрана библиотека `docx`.
- Адресные данные и файлы будут храниться в `tariff_details` и Supabase Storage.
- Требуется создать бакет `proposals` в Storage.
- Проект создан с использованием Lovable (Vite, React, shadcn/ui, Tailwind CSS).
- Для запуска требуется Node.js и npm.
- В корне присутствует `bun.lockb`, возможно использование Bun.
