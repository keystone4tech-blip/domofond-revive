# Папка медиафайлов

Эта папка предназначена для хранения всех медиафайлов проекта "Домофондар".

## Структура папок

### /images
Изображения для сайта, разделенные по категориям:

- `/products` - Изображения продуктов и услуг
- `/projects` - Фотографии выполненных работ
- `/team` - Фотографии сотрудников
- `/other` - Другие изображения

### /documents
Документы для сайта:

- `/contracts` - Договоры и контракты
- `/manuals` - Руководства и инструкции
- `/certificates` - Сертификаты и лицензии
- `/other` - Другие документы

### /videos
Видеофайлы для сайта

## Использование

Для доступа к файлам из кода используйте относительные пути от корня проекта:

```
/media/images/products/example.jpg
/media/documents/manuals/guide.pdf
/media/videos/demo.mp4
```

Все файлы в этой папке будут доступны через веб-сервер как статические ресурсы.
