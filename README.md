# Домофондар — Профессиональное обслуживание домофонных систем

Информационная система и личный кабинет абонента для компании **ООО «Домофондар»** (г. Краснодар).
Проект разработан для автоматизации работы с клиентами, ведения учета заявок (FSM), расчета тарифов и интеграции с платежными системами.

---

## 🚀 Основные возможности
* **Личный кабинет абонента**: просмотр баланса, верификация адреса (улица, дом, квартира, этаж), история обращений и оплата.
* **Интеграция с платежным шлюзом**: прямая онлайн-оплата услуг через портал банка «Кубань Кредит» с автозаполнением всех параметров плательщика.
* **FSM-система (Field Service Management)**: управление нарядами, задачами, мастерами и расписанием выездов.
* **Административная панель**: управление новостями, акциями, расчетами калькулятора, пользователями и блоками сайта.
* **Умный автокомплит адресов**: нечеткий поиск улиц и домов с Dice-коэффициентом сходства биграмм для минимизации опечаток.

---

## 🛠️ Технологический стек
* **Frontend**: React, Vite, TypeScript, Tailwind CSS, shadcn/ui, React Query.
* **Backend / API**: Node.js, Express, PostgREST (прямой REST API к PostgreSQL).
* **База данных**: PostgreSQL (схема из 42 таблиц для полного учета бизнес-процессов).
* **Контейнеризация**: Docker и Docker Compose.

---

## 💻 Локальная разработка

### Требования
* Node.js (версии 18 и выше)
* Docker и Docker Compose (для запуска локальной базы данных и API)

### Запуск проекта
1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите dev-сервер:
   ```bash
   npm run dev
   ```
   Сайт будет доступен по адресу `http://localhost:8080`.
3. Для запуска локального окружения СУБД и API выполните:
   ```bash
   docker compose up -d
   ```

---

## 🛳️ Продакшен деплой (VPS)
В проекте настроена автоматическая сборка фронтенда через Nginx и Node.js бэкенда в Docker-контейнерах.

Для ручного обновления и сборки контейнеров на сервере:
```bash
cd /opt/domofond_project
docker compose build frontend
docker compose up -d frontend
```
